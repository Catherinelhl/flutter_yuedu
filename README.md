# flutter_yuedu

自己的 Flutter 练习项目。（可能不会更新解决 Bug）

### 实现功能
1. UI（射鸡师认可的UI细粒度）
2. Provider状态管理  
3. 首页Sqlite数据缓存
4. App更新功能，带进度的下载apk安装包(这个功能网上应该还没有Flutter版本的)。
5. 统一请求的Api处理。
